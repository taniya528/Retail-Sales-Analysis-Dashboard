# Retail-Sales-Analysis-Dashboard

## Description
A data analytics project that analyzes Retail Sales data using Python, Pandas, Matplotlib and seaborn. The dashboard provides insights into sales performance, profit distribution, customer segments, regional performance, discounts and top-selling products.

Performed data cleaning, exploratory data analysis (EDA), and created visualizations to identify sales trends, profit drivers, customer segment performance, and the impact of discounts on profitability. Generated actionable business insights through a comprehensive dashboard built from retail transaction data
